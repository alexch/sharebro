UI Fixups

  • New Google Reader Rectifier -- Chrome Extension that fixes some whitespace and layout issues. Sharing still gone.
  • Google Reader Plus Theme fixed with a userstyles.org plugin -- i think he "fixed" it a bit too much but it's a start
  • Google Reader Share -- puts a 'Share' button inside the new Reader that adds the current item to your personal RSS feed, served on the author's web site
  • Google Reader Compact -- userscript by lipsumar

Replacement Readers

  • HiveMined -- Francis Cleary is writing a new RSS Reader with old-school sharing built in
  • Tiny Tiny RSS -- requires you to run your own Apache web server
  • HiveMined -- still pre-alpha
  • NewsBlur -- pretty slick, but still no social features. US$12-36 (your choice)/yr for premium
